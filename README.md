# KZero Inspire Platform

This repository contains the source code for the KZero Inspire platform, a centralized open-source hub for campus resources.

## Project Structure

- **frontend/**: The React + Vite web application (formerly `web-app`).
- **backend/**: Server-side logic and API (To be implemented).
- **reference/**: Original design assets and reference files.

## Getting Started

### Frontend
1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
